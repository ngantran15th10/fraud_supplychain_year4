# ============================================================
# CÁC CỘT QUAN TRỌNG TRONG DATASET
# ============================================================

## 1. CỘT VỀ KHÁCH HÀNG (CUSTOMER):
- Customer Id              → ID định danh khách hàng (20,652 unique)
- Customer Fname           → Tên
- Customer Lname           → Họ
- Customer Email           → Email
- Customer City            → Thành phố
- Customer State           → Bang/Tỉnh
- Customer Country         → Quốc gia
- Customer Street          → Địa chỉ đường
- Customer Zipcode         → Mã bưu điện
- Customer Segment         → Phân khúc khách hàng
- Customer Password        → Mật khẩu

## 2. CỘT VỀ SẢN PHẨM (PRODUCT):
- Product Name             → Tên sản phẩm (118 unique)
- Product Card Id          → ID sản phẩm (118 unique)
- Product Price            → Giá sản phẩm (75 mức giá)
- Product Category Id      → ID danh mục
- Product Description      → Mô tả sản phẩm
- Product Image            → Hình ảnh
- Product Status           → Trạng thái sản phẩm
- Category Name            → Tên danh mục
- Department Name          → Tên phòng ban/bộ phận

## 3. CỘT VỀ GIÁ TRỊ ĐƠN HÀNG (SALES/AMOUNT):
- Sales                    → Tổng doanh thu của đơn hàng ★
- Order Item Total         → Tổng giá trị của item ★
- Order Item Quantity      → Số lượng sản phẩm
- Order Item Product Price → Giá sản phẩm trong đơn
- Product Price            → Giá niêm yết
- Benefit per order        → Lợi nhuận từ đơn hàng
- Order Profit Per Order   → Profit của đơn
- Sales per customer       → Doanh thu trên khách hàng
- Order Item Discount      → Giảm giá
- Order Item Discount Rate → Tỷ lệ giảm giá
- Order Item Profit Ratio  → Tỷ lệ lợi nhuận

## 4. CỘT VỀ GIAN LẬN (FRAUD/STATUS):
- Order Status             → Trạng thái đơn hàng ★
  * SUSPECTED_FRAUD: 4,062 cases (2%)
  * CANCELED: 3,692 cases (2%)
  * PAYMENT_REVIEW: 1,893 cases (1%)
  * COMPLETE, PENDING_PAYMENT, PROCESSING, PENDING, CLOSED, ON_HOLD

- Late_delivery_risk       → Rủi ro giao hàng trễ ★
  * 1 = Có risk: 98,977 cases (55%)
  * 0 = Không risk: 81,542 cases (45%)

- Delivery Status          → Trạng thái giao hàng
  * Late delivery: 98,977 (55%)
  * Shipping canceled: 7,754 (4%)
  * Advance shipping: 41,592 (23%)
  * Shipping on time: 32,196 (18%)

- Benefit per order        → Nếu âm có thể là fraud
  * Âm: 33,784 cases (19%)

## 5. CỘT VỀ PHƯƠNG THỨC THANH TOÁN (PAYMENT):
- Type                     → Loại thanh toán ★
  * DEBIT
  * PAYMENT
  * TRANSFER
  * CASH

- Order Item Cardprod Id   → ID thẻ sản phẩm
